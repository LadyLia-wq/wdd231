Student Tech Resource Hub - Site Plan

Site Name

Student Tech Resource Hub

This name was selected because the website serves as a central hub where students can access technology and programming learning resources.

Optional domain: studenttechhub.org

Site Purpose

The website provides organized technology learning resources for students. It includes tutorials, recommended tools, beginner guides, and useful learning links. The site also dynamically displays resources using JSON data and allows students to submit feedback through a form.

Scenarios

  • Where can I find beginner-friendly programming tutorials?
  • What tools or websites can help me start learning web development?
  • How can I submit feedback or ask for learning support?

Color Scheme

  • Dark Blue (#0a2540): Used for headers, navigation, and footer.
  • Bright Cyan (#00b4d8): Used for accents, borders, and highlights.
  • Light Background (#f5f7fa): Used for page background.

The colors above are applied throughout this document.

Typography

  • Poppins: Used for headings and titles.
  • Roboto: Used for body text and paragraphs.

These fonts provide a modern, clean, and readable interface suitable for technology learning websites.
